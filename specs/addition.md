# 덧셈 (Addition) 명세

## Why (왜 필요한가)

사용자가 두 수를 더하는 기본 산술 연산 기능을 요청했다. 계산기, 수식 처리 등 다양한 상위 기능의 기반이 되는 핵심 연산이다.

## What (무엇을 하는가)

두 개의 숫자를 입력받아 그 합을 반환한다.

## 제약 조건

- 입력값은 JavaScript/TypeScript의 `number` 타입을 따른다
- 정수 및 부동소수점 수 모두 처리한다
- 구현은 순수 함수(pure function)로 작성한다 (부작용 없음)

## 입력 / 출력

- 입력: 두 개의 숫자 (`a: number`, `b: number`)
- 출력: 두 수의 합 (`number`)

## 엣지 케이스

- 두 수가 모두 0일 때 → 0 반환
- 음수 포함 연산 (예: -1 + 1 = 0, -3 + -2 = -5)
- 부동소수점 수 (예: 0.1 + 0.2)
- 매우 큰 수 (Number.MAX_SAFE_INTEGER 범위 내)

## 완료 기준 (Acceptance Criteria)

- [ ] `add(1, 2)`는 `3`을 반환한다
- [ ] `add(0, 0)`은 `0`을 반환한다
- [ ] `add(-1, 1)`은 `0`을 반환한다
- [ ] `add(-3, -2)`는 `-5`를 반환한다
- [ ] `add(0.1, 0.2)`는 부동소수점 오차 범위 내에서 `0.3`에 근사한다
- [ ] `add(Number.MAX_SAFE_INTEGER, 0)`은 `Number.MAX_SAFE_INTEGER`를 반환한다
